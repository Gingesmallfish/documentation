# if语句

## if单分支语法结构

::: details 点我查看语法结构
```js
    /*
     if分支语句
       if(条件)
       {满足条件要执行的代码}
     */
```
:::

```js
    if (true) {
        console.log('执行语句')
    } // 输出 true
    if (3 > 5) {
        console.log('执行语句')
    } // 不输出 false
```
## if多分支语法结构
::: details 点我查看双分支语法结构
```js
    /*
         if（条件）{
            满足条件要执行的代码
         }else{
            不满足要执行的代码
        } 
     */
```
:::
